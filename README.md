# MOVILIDAD EN LATAM
Movilidad Urbana y Productividad Económica en Ciudades de Latinoamérica
Descripción del Proyecto

Este proyecto analiza la relación entre la movilidad urbana y la productividad económica en diferentes ciudades de Latinoamérica. El objetivo principal es identificar cómo factores como el tiempo de traslado, la congestión vial y la accesibilidad al transporte pueden influir en la actividad económica de una ciudad.

A través del análisis de datos y visualizaciones, se busca comprender patrones que permitan explicar cómo una mejor infraestructura de movilidad puede impactar positivamente en la productividad y eficiencia urbana.

Objetivos

Analizar datos relacionados con movilidad urbana en ciudades de Latinoamérica.

Identificar patrones entre tiempos de traslado y productividad económica.

Generar visualizaciones que faciliten la interpretación de los datos.

Aplicar técnicas de análisis de datos para extraer conclusiones relevantes.

Tecnologías Utilizadas

Python

Pandas – Manipulación y análisis de datos

NumPy – Cálculos y manejo de datos numéricos

Matplotlib / Seaborn – Visualización de datos

Jupyter Notebook – Desarrollo y exploración del análisis

Proceso de Análisis

Recolección de datos: Obtención de datasets relacionados con movilidad urbana y variables económicas.

Limpieza de datos: Preparación y estructuración de la información para su análisis.

Análisis exploratorio: Identificación de tendencias y patrones relevantes.

Visualización: Creación de gráficos para representar la información de manera clara.

Interpretación de resultados: Análisis de cómo la movilidad urbana puede influir en la productividad económica.

Habilidades Aplicadas

Análisis de datos con Python

Limpieza y transformación de datasets

Visualización de datos

Pensamiento analítico

Interpretación de indicadores urbanos y económicos

Resultados

El análisis permitió observar que factores como los tiempos de traslado prolongados y la congestión pueden tener un impacto negativo en la productividad económica. Por otro lado, ciudades con mejores sistemas de transporte y mayor accesibilidad presentan condiciones más favorables para el desarrollo económico.
